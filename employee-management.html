Project Title: Employee Management

Employee Management GitHub

Overview

Description

This web application, developed using Angular and Spring Boot, serves as an Employee Management System with core CRUD (Create, Read, Update, Delete) functionalities. Through its intuitive interface, users can easily navigate the system to perform tasks such as viewing, adding, updating, and removing employee records. Angular ensures a responsive and dynamic front-end experience, while Spring Boot powers the back-end logic, ensuring smooth data operations. Whether administrators need to onboard new employees, update existing records, or remove outdated entries, the system offers a seamless process

Technology Used

  • Frontend: Angular, CSS, HTML
  • Backend: Java Spring boot
  • Database: PostgreSQL

Key Features

  • Display Employee: Allows users to view personal information and profiles of employees.
  • Add Employee: Enables users to input personal information and create profiles for new employees.
  • Update Employee: Allows users to update and edit the information and profiles of existing employees.
  • Delete Employee: Enables users to remove or delete employee profiles from the system.
