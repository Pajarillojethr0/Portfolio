Project Title: iLifestylePal

iLifeStyle GitHub

Overview

Description

iLifestylePal is your all-in-one companion for holistic wellness management. With features including profile creation, daily activity tracking, food journaling, sleep scheduling, step counting, and personalized suggestions based on your health condition, it empowers users to optimize their lifestyle choices and achieve their health goals. Whether you're looking to improve your diet, increase physical activity, or enhance sleep quality, this app provides the tools and guidance needed to lead a healthier and more balanced life.

Technology Used

  • Frontend: XML
  • Backend: Java
  • Database: Firebase
  • Authentication: Firebase Authentication, Google Authentication

Key Features

  • Profile Module: Allows users to input personal information and create profiles.
  • Home Module: Enables users to post images and videos of their daily activities and food intakes.
  • Daily Activity Module: Enables users to record their daily activities
  • Food Journal Module: Allows users to monitor their food intake.
  • Step Tracker Module: Monitors the number of steps the user took in a day.
  • Sleep Schedule Module: Enables users to set up their sleep schedules.
  • Recommendation System: Provides suggestions on the user's daily activity and food intake based on their physical and health condition, utilizing the Decision Tree Algorithm.
